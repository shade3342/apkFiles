# StockSignal APK

시그널 트레이딩(StockSignal) 디버그 빌드 APK 배포용 리포지토리.

## 설치 방법 (실제 폰)

1. 아래 파일을 폰에서 다운로드
   - [`stocksignal-debug.apk`](stocksignal-debug.apk)
2. 폰 설정에서 **출처를 알 수 없는 앱 설치 허용** (Chrome/파일 앱 등)
3. 다운로드한 APK 를 눌러 설치

> ⚠️ **디버그 빌드**입니다. 디버그 서명 키로 서명돼 있어 테스트용입니다.
> 스토어 배포용이 아닙니다.

## 파일

| 파일 | 빌드 | 비고 |
|---|---|---|
| `stocksignal-debug.apk` | debug | **성과검증 청산사유 하이브리드 + 전략 꼬리표 + 손익기반 뱃지/컬러 폴리싱** — ①성과 상세 팝업 청산사유를 백엔드 실제 사유코드(익절/손절/트레일링/추세이탈) 우선 표기·구버전은 가격추론 폴백(하이브리드). ②목록카드·상세 양쪽에 전략(프리셋) 꼬리표(🏷️) + 미청산 최고가 갱신 뱃지(🔥). ③청산 뱃지를 손익 기준으로 세분화 — 수익=**"익절"·레드**, 손실/본전=**"규칙 청산"·회색**('손절' 단어 미사용). ④'파란색 공포증' 교정: 익절(+수익) 매도 카드의 본문 하이라이트·매도가를 한국 관례대로 **레드**로(매도 액션 배지 배경은 유지). (백엔드 최고가<매도가 논리모순 픽스는 functions 배포분) / **(직전) 날씨 위젯 등락률 줄바꿈(%) 수정** — 코스피처럼 값이 긴 종목(7,000선)에서 "+5.15%"의 `%`가 다음 줄로 내려가던 문제 해결. 값+등락률을 단일 BasicText(2색)로 합치고 maxLines=1·softWrap=false·autoSize(9~12sp)로 폰 폭에 맞춰 자동 축소 → 어떤 폰 포맷에서도 한 줄 보장(1080·720 폭 실측). / **(직전) 시장 국면 판정 완화 + 날씨/장세 시간축 분리** — 시그널 홈 상단 코스피 '맑음(오늘 상승)'인데 하단 배너 '폭락장'으로 뜨던 인지 부조화 조치. ①상단 위젯에 "오늘의 증시 · 당일 등락", 하단 배너에 "시장 국면" 캡션을 넣어 **당일 등락 vs 며칠~몇 주 추세**를 시간축으로 구분. ②백엔드 국면 판정 완화(재배포됨): 20일선 아래라도 5일선 회복 시 폭락 즉시 해제→반등장(B1), 폭락은 200일선 이탈 또는 (변동성≥45% AND 20일선 대비 -10% 낙폭) 동시충족일 때만(B3, 변동성 단독 폭락 금지). 오토트레이더 인버스 트리거(30%)는 불변. (직전 빌드의 성과검증 청산 안내 매도 케이스별 문구 정확화도 포함) |
