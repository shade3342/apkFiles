# StockSignal APK

시그널 트레이딩(StockSignal) 디버그 빌드 APK 배포용 리포지토리.

## 설치 방법 (실제 폰)

1. 아래 파일을 폰에서 다운로드
   - [`stocksignal-debug.apk`](stocksignal-debug.apk)
2. 폰 설정에서 **출처를 알 수 없는 앱 설치 허용** (Chrome/파일 앱 등)
3. 다운로드한 APK 를 눌러 설치

> ⚠️ **디버그 빌드**입니다. 디버그 서명 키로 서명돼 있어 테스트용입니다.
> 스토어 배포용이 아닙니다.

## 파일

| 파일 | 빌드 | 비고 |
|---|---|---|
| `stocksignal-debug.apk` | debug | 마켓 배너 통합 UI 정리 — 구형 하락 경고 배너·중복 국면 뱃지·카드 하락장 뱃지를 폐기하고 백엔드 5단계 국면만 구독하는 **단일 다이내믹 배너**(강세 초록/횡보 회색/반등 골드/하락 주황/폭락 레드)로 통합. **시그널 홈·내 전략** 상단에만 노출(프리셋 마켓·조건식 화면은 배너 제거). (직전 어드민 '익명' 오표시 수정·5단계 뱃지 기능 포함) |
