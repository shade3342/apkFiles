# StockSignal APK

시그널 트레이딩(StockSignal) 디버그 빌드 APK 배포용 리포지토리.

## 설치 방법 (실제 폰)

1. 아래 파일을 폰에서 다운로드
   - [`stocksignal-debug.apk`](stocksignal-debug.apk)
2. 폰 설정에서 **출처를 알 수 없는 앱 설치 허용** (Chrome/파일 앱 등)
3. 다운로드한 APK 를 눌러 설치

> ⚠️ **디버그 빌드**입니다. 디버그 서명 키로 서명돼 있어 테스트용입니다.
> 스토어 배포용이 아닙니다.

## 파일

| 파일 | 빌드 | 비고 |
|---|---|---|
| `stocksignal-debug.apk` | debug | **성과검증 상세 시트 잘림 수정 + 문구 말투 통일** — ① 성과검증 항목(청산 완료 건) 상세 타임라인이 중간에 멈춰 하단이 잘리던 문제 수정(시트 전체 확장). ② 앱 설명문 말투를 전부 합니다체(`~입니다/~습니다`)로 통일. (이전 빌드의 **DB 무손실 마이그레이션** — 앱 업데이트 시 시그널 홈 알림·성과검증·설정 보존 — 포함) |
