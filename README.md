# StockSignal APK

시그널 트레이딩(StockSignal) 디버그 빌드 APK 배포용 리포지토리.

## 설치 방법 (실제 폰)

1. 아래 파일을 폰에서 다운로드
   - [`stocksignal-debug.apk`](stocksignal-debug.apk)
2. 폰 설정에서 **출처를 알 수 없는 앱 설치 허용** (Chrome/파일 앱 등)
3. 다운로드한 APK 를 눌러 설치

> ⚠️ **디버그 빌드**입니다. 디버그 서명 키로 서명돼 있어 테스트용입니다.
> 스토어 배포용이 아닙니다.

## 파일

| 파일 | 빌드 | 비고 |
|---|---|---|
| `stocksignal-debug.apk` | debug | **성과검증 청산 안내 문구 매도 케이스별 정확화(로컬 추론)** — 청산 완료 성과 상세의 하단 '자동 청산 안내'가 손절·익절·매도조건 청산에도 전부 "트레일링 스탑"으로 고정 표시되던 문제 수정. 서버 필드 추가 없이(동기화 트래픽 0) 진입가·최고가·매도가만으로 🎯익절 / 📉트레일링 스탑 / 🛡️손절 / 📐매도 시그널을 로컬 분류해 사유에 맞는 문구·색으로 표시(문장은 실제 수치 근거라 사실과 어긋나지 않음). (직전 빌드의 성과검증 유령 카드 삭제-반영 동기화 + 런처 아이콘 위치 보정도 포함) |
