# StockSignal APK

시그널 트레이딩(StockSignal) 디버그 빌드 APK 배포용 리포지토리.

## 설치 방법 (실제 폰)

1. 아래 파일을 폰에서 다운로드
   - [`stocksignal-debug.apk`](stocksignal-debug.apk)
2. 폰 설정에서 **출처를 알 수 없는 앱 설치 허용** (Chrome/파일 앱 등)
3. 다운로드한 APK 를 눌러 설치

> ⚠️ **디버그 빌드**입니다. 디버그 서명 키로 서명돼 있어 테스트용입니다.
> 스토어 배포용이 아닙니다.

## 파일

| 파일 | 빌드 | 비고 |
|---|---|---|
| `stocksignal-debug.apk` | debug | **성과검증 유령 카드 제거(삭제 반영 동기화) + 런처 아이콘 위치 미세조정** — ① 성과검증 델타 동기화가 upsert 전용이라 서버에서 삭제된 유령 카드가 폰에 남던 문제 수정: '열린(미청산)' 카드의 서버 존재 여부를 확인해 사라진 카드는 로컬에서도 제거(밤샘 보유 종목이 다음날 재발동 시 생기던 유령 성과검증 카드가 이 빌드로 폰에서도 정리됨). ② 런처 아이콘 V자·비콘 위치 미세 보정. (백엔드 근본 수정=밤샘 보유분 재-매수 알림/유령 카드 미생성은 서버 배포 완료) |
